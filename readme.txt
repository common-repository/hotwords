Official plugin to integrate Hotwords into your Wordpress.

== Description ==

This plugin facilitates the integration of Hotwords in Wordpress, and thus, start 
monetizing the website.

To use this plugin, an affiliate code is necessary. For this you need to contact us 
by: https://www.hotwords.com/es/afiliados

== Installation ==

Upload the Hotwords Official plugin to your blog, and Activate it.
After that, you have to configure the plugin by entering your 'affiliate code'.

If you do not have the 'affiliate code', you can get it here: https://www.hotwords.com/es/afiliados
